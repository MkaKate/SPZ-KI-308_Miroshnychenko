Програма для безпечного віддаленого доступу до комп'ютера
Репозиторій містить програму, що забезпечує безпечний віддалений доступ до комп'ютера за допомогою VPN-підключення. Програма створює захищений канал зв’язку, запобігаючи несанкціонованому доступу та перехопленню даних.

1. Встановлення захищеного VPN-з'єднання для безпечного віддаленого доступу.
2. Аутентифікація користувача перед підключенням (логін та пароль або ключі безпеки).
3. Шифрування переданих даних із використанням сучасних алгоритмів (AES-256).
4. Вибір VPN-сервера із запропонованого списку або введення власного сервера.
5. Автоматичне повторне підключення у разі розриву з'єднання.
6. Моніторинг з'єднання та виявлення підозрілої активності.
7. Журнал активності, що містить інформацію про підключення, помилки та безпекові події.
8. Графічний інтерфейс для зручного налаштування та керування підключенням.
9. Сповіщення користувача про статус підключення, помилки або загрози.
10. Можливість зміни параметрів безпеки (тип шифрування, протоколи, автозапуск VPN при старті системи).
